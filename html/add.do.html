add.do {pathways}R Documentation

add.do

Usage

add.do(x = x, h = h, col = col, .order = NA, begin = 0, position = "fill", scale_height = 1, scale.col = NA)

Arguments

x

A numeric vector or a character,is circlize current sector index ,using get.current.sector.index() to reading

h

A numeric vector or a character. If it is a character, will scale to the height of heatmap.

col

A numeric vector or a character. If it is a character, will scale to the height of heatmap. Gradient

.order

A vector color or histogram and sorting of the heatmap

begin

Starting position

position

position=c("fill","stack"), "stack" is Stacked histogram ,"fill is percentage

scale_height

Standardized height

scale.col

A color vector, name is String or number

Author(s)

dongbin wang,chao chen
